# Xindus Landing Site

Multi-page website for Xindus Technologies Pvt. Ltd.

## Pages
| File | URL |
|------|-----|
| index.html | / — Home |
| about.html | /about — About Us |
| calculators.html | /calculators — Rate Calculators |
| blogs.html | /blogs — Blog |
| track.html | /track — Track Shipment |
| 404.html | — Error Page |

## Deploy on Vercel
1. Go to vercel.com → New Project → Upload folder
2. Vercel auto-detects static HTML — no config needed
3. Done in ~15 seconds

## Deploy on GitHub Pages
1. Push all files to GitHub repo root
2. Settings → Pages → Source: main branch / root
3. Live at https://yourusername.github.io/repo-name/
